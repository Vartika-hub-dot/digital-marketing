SEO

SEO is the process of optimizing content,technical setup, and reach of our website so that our pages appear at the top of a search Engine result for specific keywords.The ultimate goal is to attract visitors to our website when they search for product or service related to our business.It is a set of technical and content practices aimed at aligning a website page with a search engine's ranking algorithm so it can be easily found algorithm so it can be easily found, crawled, indexed, and surfaced in the SERP for relevant queries. It is about making improvements to your website's structure and content so it's pages can be discovered by people searching for what you have to offer, through search engines.
There are three types of SEO:-

1.ON-PAGE SEO

On-page SEO including some contents-

  1. keywords Research
  2. Content creation(Quality,length,Structure)
  3. Meta tags(Title, Discription, Keywords)
  4. Header/heading tag(h1,h2,h3)
  5. Image optimization
  6. Internal linking
  7. Mobile-friendliness
  8. Page speed Optimization
  9. XML sitemaps

2.OFF-PAGE SEO

Off pages SEO involves optimization external factorts to improve search engine ranking.It includes some key off-pages SEO-

  • Link building
  • Social media marketing
  • Content Marketing

  • Guest Blogging
  • Review Management
  • Social marketing

3.TECHNICAL SEO

It is the process of optimizing a website's technical aspects to improve its search engine ranking. This goal is to mak it easier for search engine to crowl and index a wbsite's pages. This can lead to increased website traffic.

Techical SEO includes

a.Page speed:-Making a website faster so it's easier to navigate

SEARCH ENGINE

Search engine are the software program that provide information according to the user query.It finds various websites or web pages that are available on the internet and give related result according to the Search. such as:-google, bing, duckduckgo.

How works seach Engine?
There are 3 ways to search engine crawling, indexing, ranking
1.Crawling:Search engine have a number of computer programs that are responsible for finding information i.e, publicly available on the internet the craler scans the web and create a list of all available websites.
They visit each website and by reading html code. then try to understand the structure of the page, the type of the content and when it was created, updated, many more.
2.Indexing:Information identified by the crawler needs to be organized, sorted and stored so that it can be processed later by the ranking algorithm. Search engine don't store all the information in our index, but they keep things like the title and discription of the page.
The process of storing and optimizing and organizing the information found on web pages. This includes analyzing the page's text, images and videos and adding page to the search engine.
3.Ranking:Ranking is the position by which our website is listed in any search engine.
